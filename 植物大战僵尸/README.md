# Mini Plants vs Zombies (HTML)

This is a lightweight browser game inspired by Plants vs Zombies, built with plain HTML, CSS, and JavaScript.

## Features

- 5x9 lane board
- Plant cards: `Sunflower`, `Peashooter`, `Wallnut`
- Sun economy with sky drops and sunflower generation
- Zombie waves with increasing pressure
- Projectile combat and plant/zombie HP bars
- Lose condition (zombies break through 3 lives)
- Win condition (survive 180 seconds)

## Run

1. Open `index.html` directly in a browser.
2. Select a plant card and click tiles to place plants.
3. Click suns to collect resources.
4. Survive until timer reaches 180 seconds.

## Tech

- HTML5
- CSS3
- Vanilla JavaScript
